================================================================================

1. Open Command Prompt (cmd.exe)

2. Navigate to project folder:

3. Initialize Git and push to GitHub by running these commands:

   https://share.streamlit.io/YOUR_USERNAME/CHRO-Strategic-Insight-Engine/main/dashboard.py

================================================================================
❓ COMMON QUESTIONS
================================================================================

Q: What is my GitHub username?
A: The name you use to log into GitHub.com

Q: Where do I find the YOUR_USERNAME to use?
A: When you first created your GitHub account, you chose a username.
   It appears in the top-right corner of GitHub.com when you're logged in.

Q: I made a typo in the GitHub URL, can I fix it?
A: Yes, open Command Prompt in the project folder and run:
   git remote set-url origin https://github.com/CORRECT_USERNAME/CHRO-Strategic-Insight-Engine.git
   Then: git push -u origin main

Q: How do I update the data monthly?
A: Add new data files to the 'data' folder and run:
   git add data/
   git commit -m "Add data for new month"
   git push origin main
   Streamlit Cloud will auto-update in 2-3 minutes.

Q: Can I edit the dashboard after deployment?
A: Yes, edit files on your computer, then:
   git add .
   git commit -m "Your change description"
   git push origin main
   The live dashboard will update automatically.

Q: Who can edit the dashboard?
A: Only people you add as GitHub Collaborators with "Write" permission.
   Others can only view (read-only).

Q: What if something breaks?
A: Go to Streamlit Cloud dashboard:
   - Click on your app
   - Click "Manage app"
   - Select "Reboot app"

   If still broken, check the "Logs" tab for error messages.
